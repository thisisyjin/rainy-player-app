## 🎧 주요 기능

> - 사용자가 설정한 시간만큼 오디오 재생.
> - Audio 태그를 JS로 제어함. (play, pause, volume 등)
> - Date 객체를 활용한 날짜 렌더링
> - 비동기 제어 (setInterval / clearInterval)
> - Portrait / LandScape 변환 가능
